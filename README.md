# Aplicación para buscar algun Gif 

## Aplicación hecha en REACT utilizando la API de GIPHY


***Como Empezar*** 

>Clonar el Repo
`git clone https://github.com/AngelVelay/Gif-App/`

> Instalar las dependencias necesarias 
`npm install `

> Ejecuta el comando para ver la app 
`npm start`
